## Περιγραφη

Αυτη η εργασια προσομοιωνει ενα εστιατοριο χρησιμοποιωντας threads, mutexes και semaphores.  
Υπαρχουν γκρουπ πελατων που φτανουν τυχαια και ενας σερβιτορος που προσπαθει να τους βαλει σε τραπεζια.

Καθε ομαδα ειναι ενα ξεχωριστο thread. Οταν ερχεται, δηλωνει ποσα ατομα ειναι, περιμενει αν δεν υπαρχει διαθεσιμο τραπεζι, καθεται οταν ο σερβιτορος της δωσει τραπεζι, "τρωνε" για λιγο και μετα φευγουν.  
Ο σερβιτορος ειναι επισης thread και ελεγχει ποια ομαδα μπορει να καθισει, η περιμενει μεχρι να ελευθερωθει καποιο τραπεζι.

Χρησιμοποιησα semaphores για τον συγχρονισμο μεταξυ του σερβιτορου και των ομαδων, και mutexes για να προστατεψω την προσβαση στα τραπεζια.
